题意：给定 $k$，$n$，$w$，求至少需要多少钱才能买到 $w$ 个香蕉，其中第 $i$ 个香蕉需要 $i \times k$ 钱。

注意不需要借钱的时候就不用借了，直接输出 $0$ 即可。

```c++
#include<bits/stdc++.h>
using namespace std;
int k,n,w;
int main(){
	cin>>k>>n>>w;
	int _pay=w*(w+1)/2*k;
	cout<<max(_pay-n,0)<<endl;
	return 0;
}
```
